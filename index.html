About the data — what's available

This dashboard is computed live from the open, MIT-licensed openfootball/worldcup.json dataset (served via the jsDelivr CDN with CORS enabled). Each edition provides match-level data: round/matchday, date, kickoff, both teams, full-time & half-time score, and every goal (scorer, minute, penalty / own-goal flags), plus group, stadium, team metadata and (for 2026) squads.

FIFA also publishes an official read-only API — FAPIs (“Give Voice to Football”) at givevoicetofootball.fifa.com/api/v1 official. It is richer (live timelines, line-ups, coaches, per-player season stats) but is protected by an Akamai WAF that blocks non-browser clients, so it isn't used as this dashboard's live feed.
